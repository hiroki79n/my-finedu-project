# 🎯 FinGo - 金融教育アプリ

**Learn Finance Through Adventures - 冒険を通じて学ぶ金融教育**

Duolingo風のゲーミフィケーション学習体験を金融教育に応用した次世代型教育アプリケーション。Cloudflare Pages + Hono + React + Framer Motionで構築。

**開発環境**: https://3000-ia2yj8h4ykytgt2jhh0x0-ea026bf9.sandbox.novita.ai

**GitHub**: https://github.com/hiroki79n/my-finedu-project

## ✅ 実装済み機能

### **Phase 1: コア機能 - 完了** ✅

#### 1. **ホーム画面（学習パス）** ✅
- **Duolingo風の山登りデザイン**
  - 章ごとのノードが山道に沿って配置
  - 現在の章が最も大きく表示（スケール1.2倍）
  - 遠くの章ほど小さく表示（最小0.7倍）
  - 進捗に応じてCrownアイコン（👑）を表示
- **ダークモード + ターコイズブルー配色**
  - 背景: スレートグレーのグラデーション
  - アクセント: ターコイズ/シアン系（#14B8A6, #06B6D4）
  - テキスト: 高コントラストで視認性重視
- **Level 1: 6章を実装**
  - Chapter 101: 交換の不便さ（物々交換）
  - Chapter 102: お金の発明（貨幣の誕生）
  - Chapter 103: 価値の保存（貯蓄）
  - Chapter 104: 価値の測定（インフレ）
  - Chapter 105: 市場の意味（需要と供給）
  - Chapter 106: 労働の対価（給料）
  - 各章に4-5個のレッスンを配置

#### 2. **クイズ画面** ✅
- **4択問題形式**
  - タップして選択、ボタンで回答
  - 正解時: 緑色のフィードバック + 解説
  - 不正解時: 赤色のフィードバック + 解説
- **Chapter 101のクイズ実装済み**
  - 22問のクイズデータ（物々交換、交換の不便さなど）
  - API統合完了: `/api/quiz/chapter/:chapterId/lesson/:lessonId`
  - 回答送信API: `/api/quiz/answer`
- **リザルト画面**
  - 正解数、獲得XP、スコアを表示
  - Crown レベル判定（60% = Bronze, 80% = Silver, 95% = Gold, 100% = Legendary）

#### 3. **下部ナビゲーション** ✅
- **5つのタブ**

  - 📈 市場（Market）

#### 7. **プロフィール画面（Profile）** ✅ 🆕

## 🎨 技術スタック

### フロントエンド
- **React 18**: UIライブラリ（CDN版）
- **Framer Motion 11**: アニメーションライブラリ
- **Tailwind CSS**: ユーティリティファーストCSSフレームワーク

### バックエンド
- **Hono 4.0**: 軽量Webフレームワーク
- **Cloudflare Workers**: エッジランタイム
- **TypeScript 5.0**: 型安全な開発

### デプロイ
- **Cloudflare Pages**: 静的サイトホスティング
- **Vite 5.0**: ビルドツール
- **PM2**: プロセス管理（開発環境）

## 🚀 セットアップ手順

### 1. 依存関係のインストール
```bash
npm install
```

### 2. ビルド
```bash
npm run build
```

### 3. 開発サーバー起動
```bash
# PM2で起動
pm2 start ecosystem.config.cjs

# ログ確認
pm2 logs ecomate --nostream
```

### 4. アクセス
- **ローカル**: http://localhost:3000
- **Sandbox**: https://3000-ia2yj8h4ykytgt2jhh0x0-ea026bf9.sandbox.novita.ai
